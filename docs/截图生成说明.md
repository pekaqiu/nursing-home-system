# 截图生成说明

## 1. 运行前准备

1. 启动 MySQL 服务。
2. 执行 [sql/init.sql](/D:/File/Nursing%20home/sql/init.sql)，导入数据库 `nursing_home_db` 和演示数据。
3. 启动 Tomcat。
4. 确认浏览器可以访问 `http://localhost:8080/nursing-home-system/`。

## 2. 安装截图依赖

在项目根目录执行：

```bash
npm install
npx playwright install chromium
```

## 3. 运行截图脚本

在项目根目录执行：

```bash
npx playwright test
```

脚本会自动访问本地系统，依次登录普通用户、管理员和工作人员账号，并生成论文截图。

## 5. 使用说明

1. 如果截图脚本执行失败，请先检查 Tomcat 是否正常运行。
2. 如果页面显示为空或数据不足，请重新执行 `sql/init.sql` 导入演示数据。
3. 截图脚本使用固定视口和全页截图，适合论文插图整理。
